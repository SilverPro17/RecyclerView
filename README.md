# ReciclerView

O projeto **ReciclerView** é um aplicativo Android que gerencia informações de estudantes. Ele exibe uma lista de estudantes em um `RecyclerView` e permite adicionar, editar ou visualizar os detalhes de cada estudante.

---

## **Funcionalidades**

- Exibir uma lista de estudantes com nome, número de estudante, número de telefone, e-mail e a imagem do estudante.
- Adicionar um novo estudante com informações detalhadas.
- Editar informações de estudantes existentes.
- Suporte para imagens da galeria ou de recursos locais.

---

## **Requisitos**

- Android Studio (versão 2022.1.1 ou superior).
- Android SDK versão mínima: 24 (Android 7.0).
- Conexão de internet para carregar imagens (opcional, para imagens externas).
